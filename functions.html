User Login and Account Creation

The user can create an account or log in if they already have an existing account using the website application the system is going to use. To login, the user will have to fill up the text boxes that are displayed, and then after pressing the login button the system will determine if the user is either a regular user only or an admin. If the user selected the “Create an Account Button” it is the same function as the log in the user needs to fill up the text boxes for the system to record the user’s details.


Managing Documents/Quotations

The system also implements this feature wherein the user admin can access and retrieve data via a database. The system can be able to track the data that was sent to whom the client is or even when it is created. Using the provided search box in the website application the user admin can straightforwardly search the specific data to be displayed.


Managing Messages and Users

The same function that was mentioned in managing documents/quotation, the system also implements this feature where the user can search for a specific user and have access to the conversation between them. This function is based on an email type of messaging where the user can compose a message via a provided message box and send it to the user directly who also has an account in the website application.


Document/Invoice Creation

The system’s main feature is the document/invoice creation but this function is only permitted to the user admin and able to create documents/invoice using the website application the system is going to use. The user admin will fill up the text boxes that are displayed based and are labeled according to what the user is required to type in them. The text boxes labeled in sequence are the following:
  • Date of the document/quotation is produced
  • Client Information
  • Business Name
  • Subject of the document/quotation
  • List of Items
  • The items size
  • The items name
  • The items prices
  • The items total prices
  • Terms and Conditions
  • Percent to be paid upon down payment
  • Percent to be paid upon Delivery
  • Days to be delivered
