This study seeks to analyze the benefits of automation or automated tasks today, especially to businesses. The study focuses on how to improve the document management system, generation of documents or quotations, and communication with the means of using an application and a database. The study will be implementing a proposed system that has the functionalities of allowing the manager or admin to create a document or access and retrieve files using the website application. And, also helps clients to allow to communicate straightforwardly to the manager by using the website application.
